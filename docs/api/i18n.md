# 🌐 i18n 层 API

> `error_system::i18n` ｜ 头文件目录：`error_system/i18n/`

多语言国际化模块 — 为错误码消息与子系统/模块名称提供 locale 化的查询能力，不依赖 ICU / gettext。

---

## 🏷️ locale_t

> 头文件：`error_system/i18n/locale.h`

语言区域强类型枚举，覆盖常用语言区域，替代裸字符串提供类型安全的 locale 标识。

```cpp
enum class locale_t : uint8_t {
    en_US = 0,  zh_CN = 1,  zh_TW = 2,  ja_JP = 3,  ko_KR = 4,
    fr_FR = 5,  de_DE = 6,  es_ES = 7,  ru_RU = 8,  pt_BR = 9,
    it_IT = 10, ar_SA = 11, hi_IN = 12, th_TH = 13, vi_VN = 14,
};

inline constexpr size_t LOCALE_COUNT = 15;

// locale 名称查找表（to_string / from_string / is_valid 的唯一数据源）
inline constexpr std::array<std::pair<locale_t, std::string_view>, LOCALE_COUNT> LOCALE_TABLE = {{ ... }};

[[nodiscard]] inline constexpr std::string_view to_string(locale_t locale) noexcept;   // 非法值返回 "en_US"
[[nodiscard]] inline constexpr locale_t from_string(std::string_view text) noexcept;   // 未识别返回 en_US
[[nodiscard]] inline constexpr bool is_valid(std::string_view text) noexcept;
```

> 💡 全部函数 `inline constexpr noexcept`，可用于编译期常量。新增 locale 只需在枚举与 `LOCALE_TABLE` 末尾追加，无需修改任何转换函数。

## 📚 subsystem_module_catalog_t

> 头文件：`error_system/i18n/subsystem_module_catalog.h`

子系统/模块名称多语言目录 — 维护 `(subsystem_id, module_id) → (locale → 名称)` 的两级映射，供 `to_string()` 等可读输出场景使用。

> 💡 合并自原 `subsystem_module_registry_t`（仅支持中文硬编码）。本类**不持有** locale 状态（active/default），由调用方从 `i18n_t::instance()` 获取 locale 后显式传入查询接口，避免双源 locale 不同步。线程安全（`shared_mutex` 读多写少）。

### subsystem_module_info_t

```cpp
struct subsystem_module_info_t {
    std::string subsystem_name{"未知子系统"};
    std::string module_name{"未知模块"};
};
```

### API

```cpp
class subsystem_module_catalog_t {
public:
    subsystem_module_catalog_t() = default;
    ~subsystem_module_catalog_t() noexcept = default;
    subsystem_module_catalog_t(const subsystem_module_catalog_t&) = delete;
    subsystem_module_catalog_t& operator=(const subsystem_module_catalog_t&) = delete;
    subsystem_module_catalog_t(subsystem_module_catalog_t&&) = delete;
    subsystem_module_catalog_t& operator=(subsystem_module_catalog_t&&) = delete;

    // 注册（已存在则静默跳过，保留首次注册）
    void register_subsystem_module(locale_t locale, uint16_t subsystem_id, uint16_t module_id,
                                   std::string_view subsystem_name,
                                   std::string_view module_name) noexcept;
    void register_subsystem_module(uint16_t subsystem_id, uint16_t module_id,                    // 默认 zh_CN
                                   std::string_view subsystem_name,
                                   std::string_view module_name) noexcept;

    // 查询（返回值副本，未注册返回默认值）
    [[nodiscard]] subsystem_module_info_t get_subsystem_module_info(locale_t locale,
                                                                     locale_t fallback_locale,
                                                                     uint16_t subsystem_id,
                                                                     uint16_t module_id) const noexcept;  // 指定回退
    [[nodiscard]] subsystem_module_info_t get_subsystem_module_info(locale_t locale,
                                                                     uint16_t subsystem_id,
                                                                     uint16_t module_id) const noexcept;  // 回退 zh_CN
    [[nodiscard]] subsystem_module_info_t get_subsystem_module_info(uint16_t subsystem_id,
                                                                     uint16_t module_id) const noexcept;  // 便捷，使用 zh_CN

    // 清理
    void clear() noexcept;
    size_t clear_locale(locale_t locale) noexcept;  // 返回被清除条目数量
};
```

**使用示例**

```cpp
// 注册中文（默认）与英文
catalog.register_subsystem_module(locale_t::zh_CN, 101, 1, "交易服务", "订单模块");
catalog.register_subsystem_module(locale_t::en_US, 101, 1, "Trade Service", "Order Module");

// 查询：显式 locale + 回退 locale
auto info = catalog.get_subsystem_module_info(locale_t::en_US, locale_t::zh_CN, 101, 1);
// → subsystem_name="Trade Service", module_name="Order Module"
```

## 💬 i18n_t

> 头文件：`error_system/i18n/i18n.h`

多语言消息目录单例 — 为错误码提供 locale 化的消息查询能力。

> 💡 使用 `unordered_map<locale_t, unordered_map<code_identity_t, message_t>>` 两级哈希。`shared_mutex` 读多写少场景。locale 状态不再由本类持有，统一委托给 `config::i18n_config_t`。

### API

```cpp
class i18n_t {
public:
    static i18n_t& instance() noexcept;

    // 注册消息
    void register_message(locale_t locale, error_code_t code,
                          std::string_view message) noexcept;
    size_t register_messages(locale_t locale,
                             const std::vector<std::pair<error_code_t, std::string_view>>& entries) noexcept;

    // 查询
    [[nodiscard]] std::string get_message(locale_t locale, error_code_t code) const noexcept;  // 显式 locale
    [[nodiscard]] std::string get_message(error_code_t code) const noexcept;                    // 输出 locale

    // locale 配置（委托给 i18n_config_t，保留接口以兼容旧代码）
    void set_default_locale(locale_t locale) noexcept;
    [[nodiscard]] locale_t get_default_locale() const noexcept;
    void set_active_locale(locale_t locale) noexcept;
    void clear_active_locale() noexcept;
    [[nodiscard]] std::optional<locale_t> get_active_locale() const noexcept;

    // 管理与统计
    size_t clear_locale(locale_t locale) noexcept;
    void clear_all() noexcept;
    [[nodiscard]] std::vector<locale_t> get_locales() const noexcept;
    [[nodiscard]] size_t message_count(locale_t locale) const noexcept;
};
```

### 回退顺序

```
get_message(code)（使用 i18n_config_t::resolve_output_locale()）
1. output locale 命中 → 返回 output locale 的消息
2. default locale 命中 → 返回 default locale 的消息
3. 返回空 string

get_message(locale, code)（显式 locale）
1. 指定 locale 命中 → 返回该 locale 的消息
2. default locale 命中 → 返回 default locale 的消息
3. 返回空 string
```

**使用示例**

```cpp
auto& catalog = i18n_t::instance();
catalog.set_default_locale(locale_t::zh_CN);

catalog.register_message(locale_t::zh_CN, ERR_DB_TIMEOUT, "数据库连接超时");
catalog.register_message(locale_t::en_US, ERR_DB_TIMEOUT, "Database connection timeout");

catalog.set_active_locale(locale_t::en_US);
catalog.get_message(ERR_DB_TIMEOUT);  // → "Database connection timeout"

catalog.clear_active_locale();        // 回退到 default
catalog.get_message(ERR_DB_TIMEOUT);  // → "数据库连接超时"
```

> 🔗 locale 回退路径决策树详见 [决策树 · 4](../decision_tree.md#4-i18n-消息查询回退路径)。
